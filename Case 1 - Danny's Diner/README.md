# Case 1 - Danny's Diner

## Overview
This case study analyzes customer purchasing behavior and the loyalty program of Danny's Diner using SQL.

## Dataset
- sales
- menu
- members

## Questions Solved
- Total amount spent by each customer
- Number of visit days
- First item purchased
- Most purchased item
- Most popular item per customer
- First purchase after becoming a member
- Purchase before becoming a member
- Spending before membership
- Loyalty points calculation
- First-week bonus points calculation

## Skills Used
- JOIN
- GROUP BY
- CASE WHEN
- Window Functions
- CTEs/Subqueries
- Date Functions
